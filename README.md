# YMY RPC Eklentisi

Profilinizde tatlı bir RPC görmek ve bize destek olmak istiyorsanız...

## Kurulum & Çalıştırma

1. Repo'yu indir

```
git clone https://github.com/ymyoh/ymyrpc
```

2. Projeyi çalıştır

```
cd ymyrpc
node rpc.js
```
